Instrucciones: Parcial 1

Detalles de la Evaluación

  • Materia: Matemática I
  • Docente: Prof. García
  • Tipo de Evaluación: Parcial (Con cronómetro)
  • Fecha de Vencimiento: 25/10 19:00 hs.

Reglas Importantes

  1. Duración: Tendrás un máximo de 45 minutos para completar el examen. El tiempo comienza a correr al hacer clic en "Comenzar Evaluación".
  2. Intento Único: Solo tienes un intento para rendir esta evaluación. Una vez finalizada, no podrás volver a entrar.
  3. Guardado: Recuerda usar el botón "Guardar respuesta" después de cada pregunta de desarrollo. Si cierras la pestaña sin guardar, podrías perder tu progreso.
  4. Navegación: Puedes avanzar y retroceder entre preguntas libremente dentro del tiempo asignado.
  5. Envío Automático: Si el tiempo se agota, el examen se enviará automáticamente con las respuestas que hayas guardado.

Requisitos Técnicos

  • Asegúrate de tener una conexión a internet estable.
  • Se recomienda usar un navegador actualizado (Chrome, Firefox o Edge).
